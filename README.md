# Treasure-Vault-Mini-Game

## Prerequisites

- Node.js (v14 or higher)
- npm (v6 or higher)

## Installation

1. Clone the repository:
```bash
git clone https://github.com/DimitarBoyadzhiev/Treasure-Vault-Mini-Game.git
cd vault-mini-game
```

2. Install dependencies:
```bash
npm install
```

## Run the game

```bash
npm start
o
```
